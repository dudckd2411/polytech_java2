## 오늘 배운 내용

- 컬렉션이란?
  → 데이터를 담는 크기가 자유롭게 변하는 그릇
  컬렉션은 기본형 타입을 취급할 수 없다.

*해결책 = Wrapper Class*
기본형 데이터를 객체 형태로 감싸서 참조형처럼 보이게 만드는 클래스들을 사용해야 한다.

1. int → integer
2. double → Double
3. boolean → Boolean
4. char → Character

데이터 구조에 따른 대표적인 컬렉션

1. List : 순서 대로 쌓여있는 구조 (아이템의 중복 허용)
2. Map : key와 value의 쌍으로 저장 (키의 중복 불가)
3. Set : 순서가 없는 집합(중복 불가)

## 기억할 것

배운 것과 같음

class Person2 {

        ageMap.put("홍길동", 20);
